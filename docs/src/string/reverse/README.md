## Build an algorithm to reverse a string.

### Description

<details>
<summary> details</summary>
<p>

> Build an algorithm to reverse a string.

</p>
</details>

### Examples

**Input**

```
'table'
```

**Output**

```
'elbat'
```

### Solutions

#### Recursion simple

> Time complexity: O(n).  
> Space complexity: O(n).

#### Recursion using `String.substring()`

> Time complexity: O(n);  
> Space complexity: O(n ^ 2);

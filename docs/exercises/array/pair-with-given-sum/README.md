### Task:

Write and algorithm to find a pair of integers in unsorted array with a given sum k.

### Example

Input:  
var arr = [15, 4, 9 ,3 ,2, 12, 11, 14, 21, 24, 1, 10];  
k = 25;

Output:  
true (If found)  
or  
false (If Not found)

### Options

- Brute force (two loops)  
  Time complexity: O(n^2).  
  Space complexity: O(1).

- Using _sorting_:  
  Time complexity: O(nlogn).  
  Space complexity: O(1).

- Using _Set_:  
  Time complexity: O(n).  
  Space complexity: O(n).

### Task:

Write an algorithm to check if a subarray with 0 sum exits or not

#### Examples:

Input:  
[3, 4, -7, 3, 1, 3, 1, -4, -2, -2]

Output:
true

### Options

- Naive solution  
  Time complexity: O(n ^ 2).  
  Space complexity: O(1).

- Using **Set**
  Time complexity: O(n).  
  Space complexity: O(n).
